# Tag Images

Een Streamlit applicatie voor het categoriseren van SVG bestanden.

## Beschrijving
Deze applicatie helpt je bij het organiseren en categoriseren van SVG bestanden. Je kunt:
- Meerdere SVG bestanden uploaden
- Nieuwe categorieën toevoegen
- SVG bestanden bekijken en categoriseren
- De voortgang bijhouden
- Een JSON export krijgen van alle categoriseringen

## Aan de slag

### Installatie
1. Clone deze repository
2. Installeer de vereiste packages:
```bash
pip install -r requirements.txt
```

### Gebruik
1. Start de applicatie:
```bash
streamlit run app.py
```
2. Open de webbrowser op het aangegeven adres (meestal http://localhost:8501)
3. Upload je SVG bestanden
4. Categoriseer elk bestand door op de gewenste categorie te klikken
5. Voeg indien nodig nieuwe categorieën toe via de sidebar

## Cloud Deployment
De app kan eenvoudig worden gedeployed op Streamlit Cloud:
1. Push je code naar GitHub
2. Ga naar [share.streamlit.io](https://share.streamlit.io)
3. Connect je GitHub repository
4. Deploy de app
